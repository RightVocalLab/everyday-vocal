브라우저 TTS는 기기·OS·브라우저에 따라 제공 목소리가 다릅니다.
Android Chrome: 설정 → 접근성 → TTS → 엔진 변경
iOS Safari: 설정 → 손쉬운 사용 → 말하기 → 목소리
Mac/PC: 시스템 설정 → 손쉬운 사용 → 말하기 → 시스템 음성
남성 음성이 없다면 Google Cloud TTS / ElevenLabs 연동을 추천합니다.
